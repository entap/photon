Function array_set

多次元連想配列の要素をパス形式で指定して、値を設定する

パス形式とは、要素を辿るためのキーを並べた形式である。 例えば、下記はいずれも$array['key1']['key2']['key3']の指定となる。

  • array('key1', 'key2', 'key3')
  • 'key1[key2][key3]'
  • 'key1.key2.key3'
